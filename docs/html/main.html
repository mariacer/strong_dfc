Main script for training networks¶

This script is used for training networks on a certain dataset according to a certain algorithm. It is called by the algorithm-specific scripts that call this function with the appropriate options, namely run_bp.py, run_dfa.py, run_dfc.py and run_dfc_single_phase.py.

main.run(network_type='BP')[source]¶

Run the experiment.

This script does the following:

  • parse command-line arguments

  • initialize loggers and writers

  • create datahandler

  • create network

  • train network

Parameters

network_type (str) – The type of network.

Returns

The results summary.

Return type

summary (dict)

Reproducibility¶

To reproduce results from the papers you can use the following configurations.

MNIST¶

Backpropagation¶

The following run with 40 epochs achieves around 98.17% test accuracy.

$ python3 run_bp.py --epochs=40 --clip_grad_norm=1.0 --lr="0.002" --optimizer="Adam" --adam_beta1=0.9 --adam_beta2=0.999 --adam_epsilon="0.002" --dataset="mnist"  --size_hidden="256,256,256" --hidden_activation="tanh" --initialization="xavier_normal"
